What is Document Object Model (DOM)?

The Document Object Model (DOM) is a browser-provided interface that represents a web page's structure as a tree of nodes (elements, attributes, or text). It lets developers easily access, modify, and style a webpage dynamically using JavaScript, turning static content into interactive and dynamic experiences.

What is the event delegation in the context of the DOM, and why is it useful?

Event Delegation: Event delegation is a powerful technique in JavaScript where you take advantage of the event bubbling mechanism to handle events more efficiently. Instead of adding event listeners to individual child elements, you attach a single event listener to a parent element and handle the events triggered by its child elements

Why Event Delegation is Useful:

  • Better Performance: Instead of adding multiple event listeners to individual elements, you use just one event listener for the parent. This is especially useful when dealing with large or dynamic lists of elements.

  • Handles Dynamic Elements: When new elements are added to the DOM (e.g., dynamically inserted via JavaScript), you don't need to reattach event listeners. The parent element's listener will automatically capture events from newly added child elements.

  • Simpler Code: Reduces the need for repeated event listener setup, leading to cleaner, more maintainable code.

How do manipulate an element's attributes and styles using the DOM ?

1. Manipulating Element Attributes:

We can get, set, or remove attributes of HTML elements using methods like getAttribute(), setAttribute(), and removeAttribute().
a.setAttribute(name, value): Adds or updates an attribute
b.getAttribute(name): Retrieves the value of an attribute.
c.removeAttribute(name): Removes an attribute from the element
2. Manipulating Element Styles:

We can manipulate an element's style using the style property in JavaScript. This allows me to change individual CSS properties directly.
a.element.style.property = value: Changes or adds a CSS property to the element.
b.element.style.cssText: Allows setting multiple styles at once via a string.
